DeviantPlayer

Полная инструкция с цветными картинками: https://github.com/Etignis/DeviantPlayer.

Минимальная интрукция:

  1. Создать где нибуть общую папку для музыки.
  2. Создать в ней папку с музыкальными файлами определенной тематики. Имя папки не должно содержать заглавных букв.
  3. Повторить шаг "2" для разных наборов музык. Например "боевая", "спокойная" и т.д.
  4. Скопировать в папку из шага "1" файл "_create_music_DB.bat".
  5. Запустить этот файл двойным кликом, в папке появится файл "DB.txt".
  6. На этой странице нажать в верхней панели кнопку ""
  7. В открывшемся окне нажать первую кнопку ("Загрузить файл")
  8. Выбрать недавно созданный файл "DB.txt" из 5-го пункта.
  9. Закрыть модальное окно и нажать в врехней панели кнопку "".
  10. Выбрать галочками требуемые плейлисты.

При измененнии/добавлении/удалении файлов или папок из основной папки музыки (пункт "1"), нужно будет повторить шаги с 5-го по 10-й.

Для использования звуков:

  1. Если уже есть общая папки из для музыки (шаг "1" из списка выше), добавить в нее папку с названием "!звуки".
  2. В папке "!звуки" создать папку со звуками определенной тематики. (например "взрыв"). Имя папки не должно содержать заглавных букв.
  3. Повторить предыдущий шаг для разных наборов звуков.
  4. Если в общей папке музыки есть файл "_create_music_DB.bat", запустить его. Если нет, см. инструкцию выше.
  5. Выполнить шаги 6, 7, 8 из инструкции выше.
  6. Закрыть модальное окно и нажать в врехней панели кнопку "".
  7. Выбрать чекбоксами нужные наборы звуков.

При измененнии/добавлении/удалении файлов или папок из папки "!звуки" , нужно будет повторить шаги с 4-го по 7-й.

Так как инструкция минималистичная, дальше полагайтесь на интуицию.
