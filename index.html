Полная инструкция с картинками: https://github.com/Etignis/DeviantPlayer.

Минимальная интрукция:

  1. Создать где нибуть общую папку для музыки.
  2. Создать в ней папку с музыкальными файлами определенной тематики. Имя папки не должно содержать заглавных букв.
  3. Повторить шаг "2" для разных наборов музык. Например "боевая", "спокойная" и т.д.
  4. Скопировать в папку из шага "1" файл "_create_music_DB.bat".
  5. Запустить этот файл двойным кликом, в папке появится файл "DB.txt".
  6. На этой странице нажать в верхней панели кнопку ""
  7. В открывшемся окне нажать первую кнопку ("Загрузить файл")
  8. Выбрать недавно созданный файл "DB.txt" из 5-го пункта.
  9. Закрыть модальное окно и нажать в врехней панели кнопку "".
  10. Выбрать галочками требуемые плейлисты.
